# 3. Memory-Driven Planning Engine

## Objective
Turn procedural memory into active strategic capability.

Currently procedural memory stores routines but does not actively drive planning.

## Add Goal → Plan → Memory Loop

When a goal is detected:
1. Search episodic history for relevant patterns
2. Extract matching procedural routines
3. Propose structured multi-step plan
4. Track execution as episodic events
5. Update procedural memory based on outcomes

## Result
Memory becomes operational rather than archival.

The system transitions from remembering workflows to executing and refining them.
